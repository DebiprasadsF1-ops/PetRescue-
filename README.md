# Pet Rescue and Management Portal

A full-stack web application developed using Python and Django.

## Features
- User authentication and role-based access
- Pet reporting and management system
- Django ORM-based database design
- Responsive UI using HTML, CSS, and Bootstrap

## Tech Stack
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Project Type
Academic / Internship Project (Infosys Springboard Virtual Internship)
